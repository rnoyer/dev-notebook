# jQuery
Most popular JS library mainly used to shorten code, when it comes to manipulating DOM.

## Basics

```js
document.querySelector("h1") > $("h1")
document.querySelectorAll("button") > $("button")

$("h1")

$("h1").css("font-size")

$("h1").addClass("font-size")

$("h1").removeClass("font-size")

$("h1").Class("font-size")
```
